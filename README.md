# nippo-gen

Gemini 2.5 Flash-Lite (`google/gemini-2.5-flash-lite`) を使って日報を生成する
ツールです。 将来的にモデルを追加・切り替える可能性があります。

## 環境変数

動作には以下の環境変数の準備が必要です。

```bash
export GOOGLE_GENERATIVE_AI_API_KEY=your_api_key
export GITHUB_PERSONAL_ACCESS_TOKEN=your_pat
```

## 公開予定

追々 `npx nippo-gen` で誰でも使える OSS として公開予定です。
